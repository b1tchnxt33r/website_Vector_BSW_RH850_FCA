Issue Handling of 3rd Party Modules

Introduction

Issues of 3rd party software integrated by Vector are not part of the issue reports sent by

Vector.

Vector is not able to report 3rd party issues due to several reasons such as:

>

NDAs between Vector and the 3rd party software vendor

>

Differing release and update cycles of the 3rd party software

>

3rd party software can be updated by the customer without involving Vector

Please follow the steps defined below to retrieve issues related to the integrated 3rd party

software.
